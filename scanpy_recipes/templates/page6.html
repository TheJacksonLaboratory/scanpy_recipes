CellView Crash Course

CellView is a web application that allows knowledge-based and hypothesis-driven exploration of processed single cell transcriptomic datasets. A preprint describing CellView is available on bioRxiv.

Access

CellView is available within JAX's network here. If that link is unavailable or you are an outside collaborator, please utilize the publicly available version here.
