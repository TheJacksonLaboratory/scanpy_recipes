Visualize gene expression per cluster

You can visualize the gene expression in a chosen cluster for one or multiple genes, under the Explore tab. There are 2 sections under the Explore tab:

  1. Expression (1 gene | multiple clusters)

    2. Use Expession to visualize the expression of one gene.

    • Enter a gene of interest and select a cluster from the drop-down list. click "Run" to display a 3 panel visual output.
      • Panel 1: 3D plot of the gene expression across all cells in all clusters.
      • Panel 2: 2D plot of the gene expression across all cells in the cluster selected above. Panel 2 also has a Download button. You can download a .csv file for that 2D plot with the gene-cell expression matrix.
        1. Click and drag over a group of cells on the 2D plot.
        2. Download Expression at the top of panel 2 to start the download.
      • Panel 3: Violin plots with gene expression for all clusters. Each violin has the total cell count for each cluster (bottom of violin plots) as well as the count of cells expressing the gene(top of violin plots).
  2. Panel Plot (multiple genes | 1 cluster)

    3. Use Panel Plot to visualize the expression of multiple genes of interest in one cluster at a time.

    • Select a cluster from the drop-down list, paste a comma separated list of genes(no spaces) and click "Run".
    • The expression of each gene in the selected cluster is shown side by side in 2D plots, giving an overview of the abundance of cells expressing each gene in that specific cluster.
Compute co-expression of genes

Visualize the co-expression of multiple genes in one or all clusters under the Co-expression tab. There are 3 sections under the Co-expression tab:

  1. AllClusters

    2. Use AllClusters to visualize expression of multiple genes across all clusters on a heatmap.

  2. Selected cells

    3. Use Selected cells to visualize expression of multiple genes in selected cells within a cluster. The Selected cells sub-section has 2 panels:

    • Panel 1: 2D plot of a cluster with gene expression across cells.
      1. Enter a gene of interest and select a cluster from the drop-down list.
      2. Run to display a 2D image of the expression of the gene in that cluster.
    • Panel 2: heatmap showing co-expression in select cells for all genes listed.
      1. Click and drag over a group of cells on the 2D image in panel 1.
      2. Paste a comma separated list of genes(no spaces) in panel 2.
      3. Run.
  3. Binarize

    4. Use Binarize To vizualize cells that co-express genes of interest. This section has 2 panels:

    • Panel 1: 2D plot of a cluster with co-expression across cells. Highlighted cells have all genes expressed as determined by a Gaussian mixture model (GMM)
      1. Select a cluster from the drop-down list
      2. Paste a comma separated list of genes(no spaces)
      3. Run.
      4. Download Expression +ve Cells in cluster to get a sheet with cells co-expressing the genes of interest for the slected cluster
    • Panel 2: table with number of co-expressing cells for all clusters.
      1. OFF: Number of cells not co-expressing genes submitted by GMM.
      2. ON: Number of cell co-expressing genes submitted by GMM.
